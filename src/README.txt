Install:
1. Copy SpareWheel.js to src/SpareWheel.js
2. Copy SpareWheel.css to src/SpareWheel.css
3. Create public/data/ and place roadhero.csv inside it
4. Add route in App.js: <Route path="/spare-wheel" element={<SpareWheel />} />
5. Import SpareWheel in App.js: import SpareWheel from "./SpareWheel";
